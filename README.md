# 📊 Data Analysis Studio

Fast, local data analysis application built with Streamlit.

## 🚀 Quick Start

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the App
```bash
streamlit run data_analysis_app.py
```

App opens automatically at `http://localhost:8501`

---

## ✨ Features

- 📊 **Overview** - Data preview, column info, quality metrics
- 🔢 **Statistics** - Mean, median, std, skewness, kurtosis
- 📈 **Visualizations** - Distributions, correlations, box plots, scatter, heatmap
- 🎯 **Insights** - Key findings, completeness, correlations
- 💾 **Export** - CSV and Excel download
